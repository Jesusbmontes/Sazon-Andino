# Sazon-Andino
Sazón Andino - Frontend Elegante  Diseño de prestigio para restaurante en Madrid. Proyecto totalmente responsivo (HTML/CSS/JS) con foco en UX. Destaca por el formulario de reservas con validaciones de negocio (horario/días), un diseño de formulario premium y un flujo de confirmación con transición suave y mensaje de gratitud en una nueva pestaña.
